# 🎮 坦克大战 (Tank Battle)

一个使用纯 HTML5 Canvas 构建的经典坦克大战网页游戏。

## 🎯 游戏玩法

- **移动**: 使用 `WASD` 或 `方向键` 控制坦克移动
- **射击**: 按 `空格键` 或 `J` 键发射炮弹
- **目标**: 消灭关卡中的所有敌方坦克
- **生命**: 玩家有 3 条命，被击中会扣血，HP 归零损失一条命

## ✨ 特性

- 🎨 Canvas 渲染，流畅的像素级动画
- 🤖 敌方 AI — 巡逻、追击、射击
- 🧱 可破坏的砖墙
- 💥 粒子爆炸特效
- ⚡ 道具系统（生命恢复、加速、快速射击）
- 📱 支持移动端触屏操作
- 🏆 关卡递进，难度递增
- 🎯 屏幕震动效果
- 🛡️ 重生无敌保护

## 🚀 运行

直接在浏览器中打开 `index.html` 即可开始游戏。

```bash
# 或者用任意 HTTP 服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

tank-battle/

## 🕹️ 操作说明

| 操作 | 键盘 | 触屏 |
|------|------|------|
| 上移 | W / ↑ | 左侧上半 |
| 下移 | S / ↓ | 左侧下半 |
| 左移 | A / ← | 左侧左半 |
| 右移 | D / → | 左侧右半 |
| 射击 | 空格 / J | 右侧屏幕 |

## 🎖️ 道具

- ❤️ **生命恢复** — 恢复 1 点 HP
- ⚡ **加速** — 提升移动速度（持续 10 秒）
- 🔥 **快速射击** — 缩短射击冷却（持续 10 秒）
